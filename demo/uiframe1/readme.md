```text
使用c++及QT（可使用QSS）做一个CT界面框架。 

0、上面是菜单栏（不是经典的下拉式，跟Microsoft Word差不多），紧接菜单中子菜单工具栏（多组按钮组成）；下面是主体界面；目前支持两种界面即可（文件界面、三维界面） 

1、文件界面：左右两边区域。左边大按钮列（固定120*48），右边操作区。 

2、三维界面：左右两边区域（8:2）。左边四视图，右边操作面板。 

3、四视图有二种布局方式，2*2方式(5:5)：左下是3D，左上/右上/右下为 Axial/Sagittal/Coronal 三正交；1&3方式（7:3)：左边3D，左上/左中/左下为 Axial/Sagittal/Coronal 三正交； 

4、操作面板可展开、可收（面板在菜单中可选（加入、隐藏）） 

5、四视图与操作面板两种类型都管理起来，视图与操作面板各有基类，并有函数设置按钮状态，有函数通知某事件（事件类型及具体结构指针） 

6、视图分右边与下边与画布，右边（滚动条，滚动画布，有放大、缩小、合适等小按钮），下边（有放大、缩小、合适等小按钮，有滚动条，滚动画布）
```
