read me
=======


setup
-----
these scripts use optional python libraries. so the easiest is to work
in a python virtual environment. 

for install virtual environment, simply execute install_venv.cmd. 

or alternatively perform...
> python -m venv venv
> venv\Scripts\activate
> pip install -r requirements.txt


launch
------
> venv\Scripts\activate


run examples
------------
change the IP address by editing the source files first. then launch the use-cases:
> python <useCaseFileName>


use-cases
---------
xrayAllowed.py..........presents the currently allowed operations
xraySimple.py...........chose fix setpoints, start and stop x-ray, wait timing based
xrayConfirm.py..........chose fix setpoints, start and stop x-ray, wait state-confirmation based
xrayGuarded.py..........enable network guard, start x-ray, poll guard, then stop polling guard, expect guard failure
xrayRotateSetpoint.py...start x-ray, change setpoint multiple times, stop x-ray

guardStartAndPoll.py....configurae and start network guard and poll forever
guardStop.py............stop network guard

supervisionSimple.py....read the current supervision summary state
supervisionDetails.py...read the current supervision detailed states
supervisionPoll.py......poll effectively the current supervision detailed states
supervisionClear.py.....attempts to clear the the user-clearable errors of supervision 

measSimple.py...........present the measurements of HV and emission

warmupConfirm.py........start short warmup (which takes around 20 minutes to complete), and wait state-confirmation based

library
-------
rpc.py..................is the library for accessing the device via websocket and jsonrpc
